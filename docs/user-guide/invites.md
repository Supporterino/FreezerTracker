# Inviting Members

FreezerMan uses QR codes and invite codes to add new members to a household. No email addresses are required -- just share the code and they're in.

## How Invites Work

1. The household **owner** generates an invite
2. FreezerMan creates a short-lived invite code and a scannable QR code
3. The owner shares the QR code or invite code with the person they want to invite
4. The invitee scans the QR code or enters the code in their app
5. They join the household as a **member**

## Generating an Invite

Only the household owner can generate invites.

1. Go to household settings or the members section
2. Select **Invite Member**
3. A QR code and a text invite code are generated

> **Note:** Each invite has a time limit. If it expires before being used, you'll need to generate a new one.

## Sharing the Invite

You have two options for sharing:

- **Show the QR code** -- the easiest method when the other person is nearby. They simply scan it with their device's camera within the FreezerMan app.
- **Share the invite code** -- send the text code via any messaging app, text message, or even verbally. The invitee can type it into their app manually.

## Accepting an Invite

### Scanning a QR Code

1. Open the FreezerMan app
2. Navigate to the join/scan section
3. Point your camera at the QR code
4. Confirm that you want to join the household

### Entering an Invite Code

1. Open the FreezerMan app
2. Navigate to the join section
3. Enter the invite code manually
4. Confirm that you want to join the household

Once accepted, the new member has immediate access to all household freezers and items.

## Invite Expiry

Invites are intentionally short-lived for security. If an invite expires before someone uses it, it can no longer be accepted. Simply generate a new invite if needed.

## Viewing Active Invites

The owner can see a list of all currently active (unused, unexpired) invites from the household settings. This helps you keep track of any outstanding invitations.

## Revoking an Invite

If you change your mind or sent an invite to the wrong person, you can revoke it before it's used. A revoked invite becomes permanently invalid -- the code and QR will no longer work. You can always generate a fresh invite afterward.
