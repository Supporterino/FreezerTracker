# Freezers & Compartments

Freezers and compartments let you model the physical layout of your freezer storage so you can quickly find where things are.

## Adding a Freezer

To add a freezer to your household:

1. Navigate to the freezer overview page
2. Select **Add Freezer**
3. Enter a **name** (required) -- for example, "Kitchen Freezer" or "Garage Chest Freezer"
4. Optionally add a **description** for extra context, like the brand or location in your home
5. Save

Any household member can add a freezer.

## Freezer Dashboard

The freezer overview page shows all freezers in your current household at a glance. From here you can:

- See how many compartments and items each freezer contains
- Navigate into a specific freezer to view its compartments and items
- Add new freezers

## Compartments

Compartments represent the physical sections inside a freezer -- drawers, shelves, door bins, or however you divide the space. Every item must be assigned to a compartment, so setting these up first makes organizing your inventory much easier.

### Adding a Compartment

1. Open a freezer
2. Select **Add Compartment**
3. Enter a **name** -- for example, "Top Drawer", "Bottom Shelf", or "Door"
4. Set its **position** to control the display order
5. Save

### Editing and Reordering Compartments

You can rename a compartment or change its position at any time. Reordering compartments changes the display order only -- it does not move any items.

### Deleting a Compartment

You can delete a compartment only if it has **no active items** in it. If the compartment still contains items, you need to either:

- **Move** the items to a different compartment, or
- **Archive** the items first

This safeguard prevents accidentally losing track of items.

> **Tip:** Before deleting a compartment, use the item filter to check if anything is still assigned to it.

## Deleting a Freezer

Only the household owner can delete a freezer.

> **Warning:** Deleting a freezer permanently removes the freezer, all of its compartments, and all items within them. This cannot be undone.
